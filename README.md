# 🇧🇷 Brasil Encantado - Portal Turístico Interativo

## 📖 Sobre o Projeto
O **Brasil Encantado** é um portal online desenvolvido para uma agência fictícia de turismo, criado com o objetivo de apresentar destinos turísticos, cultura, gastronomia e experiências inesquecíveis pelo país. Este projeto foi desenvolvido como avaliação para a disciplina de Desenvolvimento Web (HTML + CSS).

O foco principal do site é atrair turistas oferecendo uma navegação intuitiva, conteúdo visualmente atrativo e uma estrutura moderna e responsiva utilizando HTML5 semântico e CSS3.

## 🚀 Estrutura do Site
O projeto é composto por múltiplas páginas interligadas e organizadas:
* 🏠 **Início (`index.html`)**: O ponto de partida para explorar as belezas do nosso país de proporções continentais, incluindo vídeos de apresentação.
* ✈️ **Destinos (`destinos.html`)**: Um mergulho na diversidade dos destinos brasileiros, de praias paradisíacas a florestas tropicais e centros históricos.
* 🍲 **Gastronomia (`gastronimia.html`)**: Uma vitrine rica dos sabores brasileiros, como Feijoada, Açaí, Pão de Queijo e Churrasco Gaúcho.
* 📞 **Contato (`contato.html`)**: Formulário de atendimento ao cliente para dúvidas, sugestões e parcerias.

## 🛠️ Tecnologias Utilizadas
* **HTML5**: Estruturação semântica (`<header>`, `<nav>`, `<main>`, `<section>`, `<aside>`, `<footer>`).
* **CSS3**: Estilização via arquivo externo (`style.css`), uso intensivo de **Flexbox** para layouts fluídos e organização visual.
* **Multimídia**: Incorporação de `iframes` do YouTube e imagens responsivas.
